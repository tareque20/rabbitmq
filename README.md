## RabbitMQ
Example with RabbitMQ

## Install RabbitMQ
[Download and install](https://www.rabbitmq.com/download.html)


## Run Project:
```sh
$ npm install
```

## Install AMPQLib
```sh
$ npm install amqplib
```

# Test Hello World Example
## Run send
```sh
$ npm run send
```
## Run Receive
```sh
$ npm run receive
```

# Test Work Queues
## Run send
```sh
$ npm run new_task
```
## Run Receive
```sh
$ npm run worker
```
